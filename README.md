# A Simple Image Annotation Tool: Tauri + Nextjs + Shadcn + Sqlite

## How to start

Change the DATABASE_URL in .env to your own path.

```bash
# Install dependencies
pnpm install
```

```bash
# Start the app
pnpm tauri dev
```

```bash
# compile the app to exe
pnpm tauri build
```
